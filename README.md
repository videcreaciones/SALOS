# SALOS
SALOS: Sistema de análisis sísmico en Python para datos de Marte y la Luna. Permite la detección y visualización de eventos mediante el procesamiento de archivos MiniSEED y CSV. Utiliza ObsPy, NumPy y Pandas para identificar umbrales de amplitud y generar gráficos detallados de segmentos sísmicos en entornos de exploración espacial.
